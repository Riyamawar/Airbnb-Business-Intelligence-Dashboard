# Airbnb-Business-Intelligence-Dashboard

## Project Overview

This project analyzes Airbnb listings, hosts, reviews, pricing, and customer ratings across major cities using Power BI. The objective was to transform raw Airbnb data into meaningful business insights through interactive visualizations and KPI tracking.

## Tools Used

* Power BI
* DAX
* Data Modeling
* Data Cleaning & Transformation

## Key Metrics

* Total Listings: 279,712
* Cities: 10
* Hosts: 182,024
* Property Types: 144
* Reviews: 537.3K

## Dashboard Features

### Overview Page

* KPI Cards
* Listing Growth Analysis
* Market Lifecycle Visualization

### Ratings Page

* Market Share by City
* Property Type Price Comparison
* Customer Rating Heatmap
* City-wise Performance Analysis

## Key Insights

* Airbnb listings grew rapidly between 2008 and 2015.
* Listing growth declined after the maturity stage and dropped significantly during COVID-19.
* Paris, New York, and Sydney contribute a major share of listings and reviews.
* Hotel rooms have the highest average price among property types.
* Several cities maintain consistently high customer satisfaction ratings.

Developed an interactive Power BI dashboard that converts Airbnb operational data into actionable business insights for performance monitoring and decision-making.
